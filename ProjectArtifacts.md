# Перечень артефактов проекта Lab-01-Esin

## Исходный код
- Program.cs — основной файл вызова модулей
- Modules/InputModule.cs — модуль ввода данных
- Modules/ValidationModule.cs — модуль проверки данных
- Modules/ProcessingModule.cs — модуль обработки и вывода данных

## Вспомогательные файлы
- README.md — описание проекта
- .gitignore — исключения Git

## Диаграммы
- UML-диаграмма модулей (UML_ModuleDiagram.png)

## Анализ модулей и зависимостей

| Модуль          | Вызываемые модули / методы          | Назначение вызова                        |
|-----------------|-------------------------------------|------------------------------------------|
| Program.cs      | InputModule.ReadInt()               | Получение данных от пользователя         |
| Program.cs      | ValidationModule.IsValidNumber()    | Проверка корректности данных             |
| Program.cs      | ProcessingModule.PrintSquare()      | Обработка данных и вывод результата      |

### Описание взаимодействия модулей
1. `Program.cs` вызывает метод `InputModule.ReadInt()` для получения данных от пользователя.  
2. Результат передаётся в `ValidationModule.IsValidNumber()` для проверки корректности.  
3. Если проверка успешна, данные передаются в `ProcessingModule.PrintSquare()` для обработки и вывода результата.  

> Данный перечень фиксирует все ключевые артефакты проекта и их зависимости, что облегчает работу с проектом и контроль изменений.
